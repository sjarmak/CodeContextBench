| Interface | WebIDL file | C++ Header | C++ Implementation |
|-----------|-------------|------------|-------------------|
| **Request** | `dom/webidl/Request.webidl` | `dom/fetch/Request.h` | `dom/fetch/Request.cpp` |

- `Request`, `Response`, and `Headers` are implemented directly under `dom/fetch/`
- `FetchEvent` is implemented in `dom/serviceworkers/ServiceWorkerEvents.h/.cpp` (as `class FetchEvent final : public ExtendableEvent`), since FetchEvent is a service worker concept
- `dom/webidl/Fetch.webidl` defines the top-level `fetch()` function interface (not a DOM interface per se, but part of the Fetch API)
