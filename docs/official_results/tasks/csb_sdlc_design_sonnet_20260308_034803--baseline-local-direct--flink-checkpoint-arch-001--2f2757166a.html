### Key Components

**JobManager Side:**
- `CheckpointCoordinator` — global orchestrator; triggers, collects ACKs, finalizes
- `PendingCheckpoint` — mutable in-flight state tracking outstanding task/coordinator ACKs
- `CompletedCheckpoint` — immutable completed checkpoint record with all state handles

**RPC Dispatch Chain:**
`CheckpointCoordinator.triggerTasks()` → `Execution.triggerCheckpoint()` → `TaskManagerGateway.triggerCheckpoint()` (RPC) → `TaskExecutor.triggerCheckpoint()` → `Task.triggerCheckpointBarrier()` → `StreamTask.triggerCheckpointAsync()` (mailbox)

1. `prepareSnapshotPreBarrier()` — pre-barrier operator work
