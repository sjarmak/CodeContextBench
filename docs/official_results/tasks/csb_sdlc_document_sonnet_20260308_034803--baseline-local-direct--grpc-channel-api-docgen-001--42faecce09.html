# gRPC C++ Channel Creation API

This document covers the public API for creating gRPC client channels and instantiating stubs in C++.

## Overview

A **channel** represents a connection to a gRPC server at a given address. Channels are created via factory functions and are thread-safe. Once created, a channel is passed to a generated stub class to make RPCs.

General pattern:

```cpp
#include <grpcpp/grpcpp.h>
#include "myservice.grpc.pb.h"

// 1. Create a channel
auto channel = grpc::CreateChannel("localhost:50051",
                                   grpc::InsecureChannelCredentials());

// 2. Create a stub from the channel
auto stub = MyService::NewStub(channel);

// 3. Make RPCs through the stub
grpc::ClientContext ctx;
MyRequest req;
MyResponse resp;
grpc::Status status = stub->MyMethod(&ctx, req, &resp);
```
