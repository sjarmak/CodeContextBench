Find all concrete struct types that implement the `runtime.Object` interface across Kubernetes API repositories.

## Background

The `runtime.Object` interface is the foundational interface for all Kubernetes API types. It is defined in the `k8s.io/apimachinery/pkg/runtime` package and requires two methods:

```go
type Object interface {
    GetObjectKind() schema.ObjectKind
    DeepCopyObject() Object
}
```

Every Kubernetes resource type (Pod, Deployment, Service, etc.) implements this interface by embedding `metav1.TypeMeta` and implementing the autogenerated `DeepCopyObject()` method.

## Repositories

Two repositories are available under `/workspace/`:

- `/workspace/api/` — kubernetes/api (standard API types: core/v1, apps/v1, etc.)
- `/workspace/apimachinery/` — kubernetes/apimachinery (runtime machinery, metav1, schema)

def find_typemeta_structs(filepath, check_fn):
    try:
        with open(filepath) as f:
            original = f.read()
    except:
        return []
    
    stripped = strip_go_comments(original)
    s_lines = stripped.split('\n')
    o_lines = original.split('\n')
    
    found = []
    i = 0
    while i < len(s_lines):
        line = s_lines[i]
        m = re.match(r'^\s*type\s+(\w+)\s+struct\s*\{', line)
        if m:
            struct_name = m.group(1)
            brace_count = line.count('{') - line.count('}')
            j = i + 1
            while j < len(s_lines) and brace_count > 0:
                brace_count += s_lines[j].count('{') - s_lines[j].count('}')
                j += 1
            orig_body = '\n'.join(o_lines[i:j])
            if check_fn(orig_body):
                found.append(struct_name)
            i = j
        else:
            i += 1
    return found


def has_api_typemeta(body):
    """metav1.TypeMeta, v1.TypeMeta, or k8s_io*.TypeMeta embedded field."""
    for line in body.split('\n'):
        stripped = line.strip()
        # Embedded field: <pkg>.TypeMeta possibly with tag
        if re.match(r'^(metav1|v1|k8s_io\w*)\.TypeMeta(\s|`|$)', stripped):
            return True
    return False
