## Overview

This migration guide covers the upgrade path for the Envoy External Processing (ext_proc) HTTP filter from **v1.30.0 (April 2024)** to **v1.31.0 (July 2024)**.

The v1.31.0 release includes **one significant breaking change** to the route cache control mechanism, a **new feature** for deferred stream closure in observability mode, and **several implementation improvements**. The main impact for users is the deprecation of the `disable_clear_route_cache` boolean field in favor of a more flexible `route_cache_action` enum field.
