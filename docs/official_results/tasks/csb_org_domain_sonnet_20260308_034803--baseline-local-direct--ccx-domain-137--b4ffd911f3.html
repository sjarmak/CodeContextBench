| 3 | `core/java/android/view/Choreographer.java` | `Choreographer.postFrameCallback()`, `doFrame()` |

**Pipeline flow:**
1. `Choreographer.doFrame()` fires on VSYNC
2. → `ViewRootImpl.performTraversals()` is called
3. → `View.measure()` → `View.layout()` → `View.draw()` on the view tree
4. → Views draw into a `RecordingCanvas`, recording operations into a `RenderNode` display list
5. → `ThreadedRenderer`/`HardwareRenderer` syncs the display list to the render thread for GPU execution
