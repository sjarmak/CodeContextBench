5. **AbstractCompiler** (`src/hotspot/share/compiler/abstractCompiler.hpp`): The abstract base class defining the common interface for both C1 and C2 compilers.

These files form the core of HotSpot's two-tier JIT compilation architecture, where CompileBroker orchestrates compilation work, CompilerThread instances execute it, and both tier compilers inherit from AbstractCompiler to provide their respective compilation strategies.
