## Grafana Database Migration Framework

The framework lives in `pkg/services/sqlstore/migrator/` and consists of:

### Core Files

| File | Purpose |
|------|---------|
| `migrator.go` | **Migration runner** — `Migrator` struct, `MigrationLog` (version tracking table), `Run`/`doMigration`/`InTransaction` (rollback handling) |
| `types.go` | `Migration` interface, `CodeMigration`, `Table`, `Index` types |
| `migrations.go` | Concrete migration types: `RawSQLMigration`, `AddTableMigration`, `AddColumnMigration`, `DropTableMigration`, etc. |
| `column.go` | `Column` type |
| `conditions.go` | `MigrationCondition` interface and implementations (`IfIndexExistsCondition`, `IfColumnNotExistsCondition`, etc.) |
| `dialect.go` | `Dialect` interface + `BaseDialect` — SQL dialect abstraction |
| `mysql_dialect.go` | `MySQLDialect` — MySQL-specific SQL generation and locking |
| `postgres_dialect.go` | `PostgresDialect` — Postgres-specific SQL generation and locking |
| `sqlite_dialect.go` | `SQLite3` — SQLite-specific SQL generation |

### Key Design Points
